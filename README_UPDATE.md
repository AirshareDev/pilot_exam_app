# v0.17.0 更新手順

1. ZIPを展開します。
2. 現在の `pilot_exam_app` フォルダへ全ファイルを上書きします。
3. 次を実行します。

```powershell
flutter pub get
flutter analyze
flutter run
```

## 模擬試験の出題仕様

- 有効科目を表示順に最大5科目使用します。
- 各科目から10問ずつ無作為抽出します。
- 5科目×10問で50問になる構成です。
- 科目数や問題数が不足する場合は、未選択の問題から補完します。
- DB全体に50問未満しかない場合は、登録済みの問題だけで実施します。

`user.db` は上書き更新されないため、購入状態、回答履歴、ブックマーク、成績は保持されます。
