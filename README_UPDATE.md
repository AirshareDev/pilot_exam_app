# v0.19.2 更新手順

このZIPの `lib` フォルダを、現在のFlutterプロジェクトの `lib` フォルダへ上書きしてください。

## 確認コマンド

```powershell
flutter pub get
flutter analyze
flutter run
```

## 確認項目

1. 模擬試験から「練習モード」を開始する
2. 回答を選択しても、その場で正誤・解説が表示されないこと
3. 回答後も選択肢を変更できること
4. 未回答のまま「次へ」を押せること
5. 右上の「中断」を押すと確認ダイアログが表示されること
6. 中断後、ホーム画面に「続きから学習」が表示されること
7. 再開すると、問題順・現在位置・回答内容が復元されること
8. 最後の問題で「採点する」を押すとまとめて採点されること

## 補足

- 本試験モードは従来どおり中断不可です。
- 中断データは既存の学習再開情報と同じ保存領域を使います。そのため、別の学習を中断保存すると、以前の中断情報は置き換わります。
